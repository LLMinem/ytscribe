# [First steps with uv](https://docs.astral.sh/uv/getting-started/first-steps/#first-steps-with-uv)

After [installing uv](https://docs.astral.sh/uv/getting-started/installation/), you can check that uv is available by running the `uv` command:

```
$ uv
An extremely fast Python package manager.

Usage: uv [OPTIONS] <COMMAND>

...
```

You should see a help menu listing the available commands.

## [Next steps](https://docs.astral.sh/uv/getting-started/first-steps/#next-steps)

Now that you've confirmed uv is installed, check out an [overview of features](https://docs.astral.sh/uv/getting-started/features/), learn how to [get help](https://docs.astral.sh/uv/getting-started/help/) if you run into any problems, or jump to the [guides](https://docs.astral.sh/uv/guides/) to start using uv.
